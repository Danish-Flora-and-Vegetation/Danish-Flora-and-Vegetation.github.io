1 Introduction

In this exercise we are characterizing the most common species found in the Danish monitoring program NOVANA (Nationale Overvågningsprogram for Vand og Natur). The data observations includes 1.855.717 samples in 2.450 sites, and 50.261 plots sampled from 2003, till 2015. This includes 1.757 that have been sampled at least once.

If we take into account how many plots a species has been detected at leas once, you can see in table 1.1 the list of the species in Denmark ordered by the number of occurrences it has, note that using the search bar you can search for only one family.
